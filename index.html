Abstract

Autonomous driving (AD) systems relying solely on onboard sensors may fail to detect distant or occluded hazards, potentially causing preventable collisions; however, existing transformer-based Vehicle-to-Everything (V2X) approaches either lack effective multimodal fusion and reasoning or struggle to meet real-time performance requirements under complex traffic conditions. This paper proposes the Real-time Edge-based Autonomous Co-pilot Trajectory planner (REACT), a V2X-integrated trajectory optimization framework for AD based on a fine-tuned lightweight Vision–Language–Action (VLA) model explicitly grounded in low-level control generation. REACT integrates infrastructure-provided hazard alerts with onboard sensor data, capturing surrounding traffic dynamics and vehicle intents through visual embeddings, interpreting numerical data from symbolic inputs, and employing contextual reasoning to predict control-aware residual action sequences. To enable robust real-time deployment on edge devices, REACT employs a Residual Trajectory Fusion (RTF) module that predicts discrete residual controls and fuses them with a nominal plan, together with edge-adaptation strategies to reduce model complexity and improve inference efficiency. Evaluated on the DeepAccident benchmark, REACT achieves state-of-the-art performance, an 84.9% collision rate reduction, a 50.6% Video Panoptic Quality, and a 0.57-second inference latency on the Jetson AGX Orin. Ablation studies validate the contribution of each input modality, architectural component, and edge adaptation strategy. These results highlight the effectiveness of lightweight control-grounded VLA reasoning for real-time cooperative planning on edge platforms and underscore the potential of language-guided residual control generation for improving traffic safety.

Keywords: Control sequence prediction · Transportation safety · Connected and autonomous vehicles · Edge computing

Key Contributions

Architecture

REACT — a V2X-integrated autonomous-driving framework built on a customized VLA model that directly generates low-level vehicle controls for safer, context-aware driving.

Methodology

A lightweight VLA reasoning pipeline that jointly grounds perception and action to predict steering and acceleration sequences, with a novel Residual Trajectory Fusion module for efficiency and accuracy.

Edge Optimization

Targeted edge-adaptation strategies — prompt optimization and parameter-efficient fine-tuning — streamline complexity and enable real-time control inference on resource-constrained devices.

Practical Applicability

Validated across diverse traffic scenarios, weather, lighting and infrastructure configurations, demonstrating robust decision-making in realistic V2X-enabled deployment.

Method

REACT runs alongside an existing autonomous-driving stack and turns multimodal V2X inputs into a stable, executable control sequence through five stages: Data Acquisition → Data Structuring → Task Alignment → Task Projection Enhancement → Residual Trajectory Fusion.

Experiment Results

Evaluated on the DeepAccident benchmark against representative V2X fusion methods. REACT delivers the best motion-prediction quality while running an order of magnitude faster than 7B-scale baselines.

Qualitative Results

Across eight accident-prone scenarios, REACT anticipates hazards earlier and produces smoother avoidance maneuvers and safer stopping behavior than DiscoNet, V2X-ViT and CoBEVT — in both direct-accident (a–d) and secondary-risk (e–h) cases.

Ablation Studies

Ablations isolate three design choices: the RTF module & CoT-SFT (chain-of-thought supervised fine-tuning), the input modalities, and edge-adaptation strategies.

Every input modality matters

Removing any input degrades performance; dropping the control sequence collapses collision-rate reduction from 84.9% to 51.7%, confirming that control-aware grounding is central to safe planning.

Takeaways. The RTF module with CoT supervision attains SFT-level accuracy (mIoU 57.85, VPQ 50.59) while reducing 10-step latency from 1.43 s to 0.57 s (>60%) at <0.1% accuracy cost. Among edge-adaptation choices, token length dominates the latency budget, while moderate BEV resolution and quantization preserve safety-critical accuracy.
